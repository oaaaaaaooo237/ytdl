# GUI Reference

Selected concept: the first generated GUI mockup, a light Windows 11-style utility interface.

Reference image: `docs/gui-reference.png`

Implementation notes:
- Use this image as the visual baseline for the desktop app.
- Keep the light theme, left navigation rail, teal primary action color, and clear three-step download flow.
- Main areas should cover URL analysis, video preview, save location, audio/video/playback toggles, format selection, download queue, history, and yt-dlp update status.
- The final app UI should use Chinese labels, even though the reference mockup text is mostly English.
