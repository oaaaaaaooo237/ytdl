# Android GUI 基准

用户于 2026-06-19 确认安卓版 GUI 采用原始 ImageGen 第三张图的布局方向。该图由用户补发确认，必须作为后续开发的权威视觉基准；不要用后补重绘图、占位图或文字描述替代。

## 基准图

- 文件：`docs/android-gui-reference-v3.png`
- 用途：Android Play 版后续 GUI 设计、实现、审计和验收的视觉基准。
- 内容：同一张图内展示下载、格式、队列、历史、设置 5 个手机页面。

## 还原要求

- 尽可能还原原始第三版图片的页面布局、视觉层级和交互骨架。
- 底部固定 5 个页面按钮：下载、格式、队列、历史、设置。
- 五个主要页面都要纳入设计和实现，不得只做首页。
- 队列页要按可滚动列表设计，并显示真实进度、速度、ETA、暂停和取消入口。
- 页面顶部要考虑 Android 状态栏和挖孔摄像头安全区。

## 配色策略

默认采用 `docs/android-gui-reference-v3.png` 中的 Material You / Bento 配色。

后续如需换色，只允许两类策略：

1. 参考 Codex 主题配色，统一设计 5 个页面。
2. 分别设计 5 个页面各自的背景色、前景色和强调色。
